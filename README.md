# Save Text

Save Text is a minimal web utility for transporting plain text across devices.

You paste or write text in the website, click **Save**, and the content becomes accessible via a unique URL. This makes it useful for quickly moving text between computers, phones, or environments without accounts or logins.

## How it works

1. Enter any plain text into the website.
2. Click **Save**.
3. Choose an identifier consisting of **6 or more characters** (letters, digits, or symbols).
4. Your text becomes available at:

```
https://save-text.github.io/<identifier>
```

Anyone with the link can view the content.

## Use cases

* Moving text between devices
* Temporarily sharing logs, snippets, or notes
* Copying text from restricted or isolated environments
* Quick sharing without authentication or setup

## Important notes

* **All saved content is public.**
* There is **no encryption, authentication, or access control**.
* Do **not** store passwords, personal data, API keys, or sensitive information.
* Identifiers are not secret and should not be treated as protection.

## Limitations

* Plain text only
* No edit history or versioning
* Content may be removed at any time
* No guarantee of permanence or availability
